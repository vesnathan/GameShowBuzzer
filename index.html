GAME SHOW SCORING SYSTEM DEMO

It's best to use Chrome for this demo.

On iPad's download "Frameless" app for a full screen browser.

If you're using Chrome on a laptop press F11 for full screen.

 
1. Open Control Screen on your main computer.
Select "Demo Show" from the saved shows box, enter the password 'password', and click "OPEN",
or, enter a new show name and password, then use the "PLAYERS" tab to add as many players as you need.
control.html
 
2. Open Scoreboard Screen.
This would be displayed on a large flat panel screen, preferrably in portrait orientation.
Use a laptop or Raspberry Pi to drive the screen.

Type the Scoreboard SCreen Code into the Control Panels "SETTINGS" tab.
scoreboard.html
 
3. Open the players score screens that you need.
Each team needs either an iPad, Laptop, or Raspberry Pi and Screen.
On the control panel PLAYERS tab add the displayed score screen ID's for the each player.
playerscreen.html
 
4. Go to the "BUTTON CONFIG" tab on the Control Panel.
Have the players go to the IP address provided using their phones, and type their teams code into the page that opens.
You can have multiple buttons per team. Players can see their button trigger on the control panel.

Once users are happy their buttons are working, select the "SCORING" tab, and select "START SHOW" to start scoring.

When a player presses their button, other players will be locked out until the first button presser's
answer is marked either correct, incorrect, or cleared.
